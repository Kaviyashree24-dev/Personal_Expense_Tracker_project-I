# UI Wireframe Design

## Objective
Design the basic layout of all pages in the Personal Expense Tracker before implementing the actual user interface.

## Wireframe Pages

### 1. Home Page (index.html)
- Project Title
- Login Button
- Sign Up Button
- Introduction Section

### 2. Login Page (login.html)
- Email Field
- Password Field
- Login Button
- Sign Up Link

### 3. Registration Page (signup.html)
- Name Field
- Email Field
- Password Field
- Confirm Password Field
- Register Button

### 4. Dashboard Page (dashboard.html)
- Navigation Menu
- Total Expenses Summary
- Quick Access Buttons
  - Add Expense
  - View Expenses
  - Categories
  - Logout

### 5. Add Expense Page (add-expense.html)
- Expense Title
- Category Dropdown
- Amount Field
- Date Field
- Add Expense Button

### 6. View Expenses Page (view-expenses.html)
- Search Box
- Expense Table
- Edit Option
- Delete Option

### 7. Categories Page (categories.html)
- Category Name Field
- Add Category Button
- Category List

### 8. Logout Page (logout.html)
- Logout Confirmation Message
- Return to Login Page

## Deliverable
Page Layouts (Wireframes) designed for all project modules.
